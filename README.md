ru:
 это клавиатура является гибридом экранной клавиатуры дял телефона и физической 65% клавиатуры для компьютера. Написана преимущественно с помощью deepseek-r1, хотя задумка собственная
 
<img width="1080" height="857" alt="image" src="https://github.com/user-attachments/assets/233956f0-191a-4f0c-9320-0bd14a0ebc27" />


комбинации:
   
  alt + space = смена раскладки

  alt + del = esc
 
  ctrl + delete/backspace = удалить слово целиком вперёд курсора / назад

  alt + t = alt + е = написать "ё"

  ctrl + t = табуляция

  ctrl + - = сдвиг курсора назад

  ctrl + = = сдвиг курсора вперёд

  alt + = = открытие NumPad'a

  alt + - = ~
  
  alt + ' = `

  ctrl/alt + любая клавиша = комбинация клавиш на усмотрение системы

клавиатуру можно доделать по своему желанию

en:
 Keyboard is hybrid screen phone keyboard and 65% keyboard for pc. It was written w/ deepseek-r1.
 
  *images from "ru"*

 combinations:
  ctrl + bs = delete word back cursor

  ctrl + del = delete word forward cursor
  
  alt + t = write "ё" (for ru lang)

  ctrl + t = tabulation

  ctrl + - = move cursor back

  ctrl + - = move cursor forward

  alt + = = open NumPad 

  alt + - = ~
  
  alt + ' = `

  alt + space = toggle language

  alt + del = esc

 u can resume dev at will
